JS 语言的基本构成、变量、数据类型、表达式、选择结构、循环结构、短路语句、函数基础；
JQuery 选择器、基本操作API、动画API、事件API、插件机制、原理分析、项目实战；
AJAX 基本编程接口、异步数据交互、模板引擎的使用、跨域的实现方案、增量加载；
JSON 语法；
HTTP协议。
